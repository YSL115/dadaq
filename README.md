# dadaq

达达趣 SyncNext 插件仓库。

| 类型 | 链接 |
| --- | --- |
| 插件订阅地址（推荐） | `syncnextPlugin://https://cdn.jsdelivr.net/gh/YSL115/dadaq@main/config.json` |
| 插件订阅地址（Raw） | `syncnextPlugin://https://raw.githubusercontent.com/YSL115/dadaq/main/config.json` |
| 插件订阅地址（Raw 代理） | `syncnextPlugin://https://ghproxy.net/https://raw.githubusercontent.com/YSL115/dadaq/main/config.json` |
| 配置直链（推荐） | `https://cdn.jsdelivr.net/gh/YSL115/dadaq@main/config.json` |
| 配置直链（Raw） | `https://raw.githubusercontent.com/YSL115/dadaq/main/config.json` |

## 文件说明

- `config.json`：插件配置入口
- `main.js`：解析与播放逻辑
